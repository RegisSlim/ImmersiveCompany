
# Immersive Company

An immersion focused mod.

Features:

-    Ladder stays extended permanently.
-    Doors can be locked with keys.
-    2 new scrap items.
-    New tools.

Planned:

-    More scrap.
-    Glowsticks.
-    Radar booster overhaul.

Tools:
- Personal Radar:
  -   Price: 400
  -   Detects scrap in the vicinity of the scanner but has a short battery life.
