
# Immersive Company

An immersion focused mod.

Features:

-    Ladder stays extended permanently.
-    Doors can be locked with keys.
-    2 new scrap items.
-    New tools.

Planned:

-    More scrap.
-    Glowsticks.
-    Radar booster overhaul.

Tools:
- Personal Radar: Costs 350 and has a short battery life.
